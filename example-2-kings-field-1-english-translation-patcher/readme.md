King's Field English Translation Patcher Example for Lib-Enigma, by Alex Free.

If you give this patcher The King's Field (Japan) bin file, it will apply the english translation (https://www.romhacking.net/translations/1067/) with save fix (https://www.romhacking.net/translations/7308/). If you run the patcher again on a bin file already patched, it will undo the patch and restore the bin file to Japanese language.

Executable name: kfeng.
