Piracy Patcher Example for Lib-Enigma, by Alex Free.

If you give this patcher The MediEvil (Europe) bin file, it will patch out the LibCrypt v1 protection. For all other PSX game bin files you give it, the patcher will look for Anti-Piracy v2 protection. If it finds it, it will apply my 'fake VC0' bypass, which will allow the game to be played on a console that does not have a non-stealth mod-chip, as long as you use a soft-mod to boot it.

Executable name: sec viewer.
