PSX Boot File Printer Example for Lib-Enigma, by Alex Free.

If you give this patcher a PSX game bin file, it will print the boot file name. For old style Japanese games that have PSX.EXE as the boot file, it will try to use the volume creation timestamp and if a match is found it will instead print out the game serial code as the boot file to more uniquely identify it.

So for example for King's Field (Japan), instead of printing out 'PSX.EXE', it will print out 'SLPS_000.17'.
